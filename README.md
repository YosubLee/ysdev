# 소프트웨어 개발 블로그
## local 실행
1. Ruby 설치 
[Ruby Download Site](https://www.ruby-lang.org/en/downloads/)
2. `jekyll bundler` 설치
````
gem install jekyll bundler
````
3 . local 실행
````
bundle exec jekyll serve
````
